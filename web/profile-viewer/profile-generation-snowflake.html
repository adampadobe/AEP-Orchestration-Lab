Snowflake IP allowlist

              

                Cloud Functions egress for this lab is pinned to a single static IPv4 via Cloud NAT. Add it to your Snowflake NETWORK POLICY before testing.
              

            

            

              Lab static egress IP
              34.58.81.28
              
            

            
CREATE NETWORK POLICY aep_orchestration_lab_policy
  ALLOWED_IP_LIST = ('34.58.81.28/32');

ALTER USER <your_user> SET NETWORK_POLICY = aep_orchestration_lab_policy;

            

              Full setup guide: docs/SNOWFLAKE_INTEGRATION.md.
