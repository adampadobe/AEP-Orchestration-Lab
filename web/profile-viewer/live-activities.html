Live activities

Send a Live Activity push in four quick steps: 1) find the person · 2) campaign, ECID, Live Activity ID, and event · 3) message & template · Travel appearance when generic · 4) Update message from fields, preview, Send to Adobe. For a brand-agnostic iOS widget, use Travel (Generic) templates with TravelLiveActivityAttributes — theme + phase drive the layout.

1 Find the person

Choose how you’re looking them up (email, ECID, etc.), enter the value, then Get profile. When a profile is found, ECID is filled in for step 2.

2–4 Build & send the unitary message

2) Campaign & recipient · 3) Message & template (JSON editor is optional and collapsed by default) · Travel appearance when the template is Travel generic · 4) Update message from fields, then Refresh preview / Send to Adobe. Uses POST /ajo/im/executions/unitary via this app’s proxy.
