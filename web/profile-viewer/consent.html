How we use your data

Data collection Allow Experience Platform to collect profile and experience data for this identity.
Data sharing Allow sharing of this profile with Adobe solutions and connected destinations per your policies.
Content personalization Use profile attributes to tailor content and offers you see across channels.

Prepare creates schema AEP Profile Viewer - Consent - Schema, dataset AEP Profile Viewer - Consent - Dataset (not enabled for Profile), and primary identity on _{tenant}.identification.core.email using your tenant Profile Core v2 mixin. In AEP, create an HTTP API streaming dataflow named AEP Profile Viewer - Consent - Dataflow for that dataset — do not enable the dataset for Profile — then paste the collection URL and Flow ID below (saved in this browser).
