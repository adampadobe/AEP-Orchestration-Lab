03

Next best path

Inside the chosen Mortgage refinance journey, the engine arbitrates across four candidate paths. Each path is a sequence of actions toward conversion. Path scoring blends intrinsic effectiveness with the customer's signal alignment, behavioural fit, and any active time-pressure context.

AJO 101 What is a conditional split? ▾

In Adobe Journey Optimizer, a journey is built on a visual canvas. A conditional split node is a fork in that canvas — incoming profiles are evaluated against a rule, and each profile is routed down one branch or another. The branches you see below (Rate calculator, Pre-approval flow, Advisor callback, In-branch consultation) are the four branches of a conditional split inside the Mortgage refinance journey. The percentage on each branch shows the share of profiles that flowed down that branch — your real metric for path effectiveness.

Why it matters: the path scoring formula above runs per profile at the split node. Two profiles entering the same journey can take entirely different branches because their signal mix tips the formula differently. That's how one mortgage journey can produce a hyper-personalised outcome for thousands of distinct members.
