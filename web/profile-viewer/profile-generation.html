🧬 Generic profile (industry-less)

      

        Skip the industry pack and stream a clean Profile with Customer Analytics fields (churn, propensity, NPS, AOV, preferred channel, gender, loyalty, language).
        Set up the schema, attach field groups and create the Profile-enabled dataset for any sandbox; then generate or update profiles via Adobe Profile Updater (/api/profile/update).
      


      
      

        
          
            ⚙
            Sandbox & streaming connection
          
          Per-sandbox setup. Run the four steps once per sandbox, then create the HTTP API dataflow in AEP and click Fetch URL & Flow ID.
        
        

          

            
Before you can generate or update generic profiles in a new sandbox:

            
    
              
  1. 1 · Create schema — Creates an empty Profile schema named AEP Lab - Generic Profile - Schema.
    2. 
              
  2. 2 · Attach field groups — Attaches Profile Core v2, Demographic Details, Personal Contact Details, Loyalty Details and a tenant AEP Lab - Customer Analytics field group, plus the primary Email identity descriptor.
    3. 
              
  3. 3 · Create dataset — Creates AEP Lab - Generic Profile - Dataset, enabled for Real-Time Customer Profile (opposite of the Consent dataset).
    4. 
              
  4. 4 · HTTP flow (instructions) — Tells you how to create the HTTP API dataflow in AEP UI. Then use Fetch URL & Flow ID to populate the connection and click Save connection.
    5.
