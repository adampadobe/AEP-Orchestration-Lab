Confirmed flow. You sign in here to manage data. For Adobe, you copy one URL — no API key and no Firebase login inside Journey Optimizer.

  1. Create or use a Firebase account on this page (email/password or guest).
  2. Enter your workspace name (for example LDAP), then click Save workspace name. This names your folder; it is not your Adobe password.
  3. Click Refresh data at least once so your workspace is copied to the endpoint Adobe will call.
  4. Copy the AJO personalization URL below. In Journey Optimizer, add a custom action, paste that URL as the endpoint, set Method to GET, and leave authentication / headers for auth empty.

Reads for that URL are intentionally open to anyone with the link (for personalization only). Do not put secrets or private data in your workspace if you use this URL.

Using this URL in Journey Optimizer (AJO)

In Journey Optimizer: Administration → Configurations → Actions → Create Action. Use the AJO personalization URL from this page, method GET, no authentication. Choose the sandbox that matches your journeys.

The ajoLookups endpoint is public-read by design. The userWorkspaces URL below is for integrations that pass a Firebase ID token.
