Live activities

Send a Live Activity push through AJO in four steps: 1) find the person · 2) enter campaign, ECID, Live Activity ID, and event · 3) load or paste JSON · 4) Apply all from fields, then Send to Adobe.

1 Find the person

Choose how you’re looking them up (email, ECID, etc.), enter the value, then Get profile. When a profile is found, ECID is filled in for step 2.

2–4 Build & send the unitary message

2) Enter the values below (they must match your campaign and device). 3) Load a template or paste JSON. 4) Use Refresh preview when you want to see the exact outgoing body, then Send to Adobe. Uses POST /ajo/im/executions/unitary via this app’s proxy.
