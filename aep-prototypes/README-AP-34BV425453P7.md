### AEP Profile

- **00 Adobe Auth** – Adobe IMS authentication
- **01 Profile Query** – Query profiles and datasets
- **02 Ingest Events** – Stream experience events to AEP
- **03 Profile Viewer** – Web app for viewing profiles, generating profiles, and event generation

See [AEP Profile/README.md](./AEP%20Profile/README.md) for setup and usage.
